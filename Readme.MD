# ShadowFox Internship Projects

This repository contains the projects completed during my internship at **ShadowFox**. It includes three independent web development projects: a **Dental Clinic Website**, an **E-Commerce Website**, and my **Personal Portfolio Website**.

## 📁 Repository Structure

```
ShadowFox-InternShip/
│

```

---

## 🦷 1. Dental Clinic Website

A responsive website built for a dental clinic, designed to showcase services, doctors, and appointment information for patients.

**Features:**
- Clean, responsive UI
- Service listing and clinic information pages
- Component-based architecture

**Tech Stack:**
- React
- Vite
- ESLint / Oxlint for code linting

**Getting Started:**
```bash
cd DentalClinic
npm install
npm run dev
```
